# 4. Sequence & Traversal Patterns

This chapter teaches how to move through sequences and nested structures while
maintaining just enough state.

## What This Chapter Should Prove

- use two pointers and sliding windows
- answer range questions with prefix sums
- preserve invariants in stack and queue algorithms
- traverse trees with DFS and BFS
- prevent repeated work with a visited set
- distinguish traversal order from result order

## Drill Order

1. Two-pointer pair sum
2. Remove duplicates from a sorted list
3. Fixed-size sliding-window maximum sum
4. Variable-size sliding window
5. Prefix-sum range queries
6. Balanced brackets
7. DFS and BFS tree traversal
8. Graph reachability with a visited set
